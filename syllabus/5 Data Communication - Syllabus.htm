Data Communication [CT-602]

Course Objective:
The objective of the course is to familiarize student with the concept of data communication, communication signals and their characteristics, transmission media and their characteristics, basics of multiplexing and switching.

  1. Introduction [4 hours]
    1. Data and Signal
    2. Analog and Digital Signal
    3. Data Representation
    4. Analog and Digital Data Communication System
    5. Transmission Impairments (Attenuation, Noise, Distortion)

  2. Signals and Systems [4 hours]
    1. Signal and Classification of Signals: Periodic and Non-periodic Signals, Deterministic and Random Signals, Energy and Power Signals, Continuous Time and Discrete Time Signals
    2. System and Basic Properties of Systems: System with and without memory, Linearity, Time Invariance, Invertibility, Casuality, Stability

  3. Signal Analysis [6 hours]
    1. Unit Impulse Function and Unit Step Function
    2. LTI System and Impulse Response
    3. Fourier Series Representation of Continuous Time Signal
    4. Fourier Transform of Continuous Time Signal
    5. Spectral Analysis of a Signal, Signal Bandwidth

  4. Transmission Media [4 hours]
    1. Electromagnetic Spectrum for Communication and Type of Propagation
    2. Guided Transmission Media: Copper Media (Twisted pair and Co-axial) and Fiber Optics
    3. Unguided Communication Bands and Antennas
    4. Unguided Transmission Media: Terrestrial Microwaves, Satellite Communication and Cellular System
    5. Data Rate Limits: Nyquist Bit Rate for Noiseless Channel, Shannon Capacity for Noisy Channel
    6. Performance of Channel: Bandwidth, Throughput, Latency, Jitter, Bit Error Rate (BER)

  5. Data Encoding and Modulation [10 hours]
    1. Baseband Communication (Analog/Digital)
    2. Data Encoding and Modulation
    3. Types of Analog Modulation: Amplitude Modulation, Frequency Modulation and Phase Modulation
    4. Pulse Modulation System: Pulse Amplitude Modulation (PAM), Pulse Width Modulation (PWM)
    5. Encoding Analog Data as Digital Signal: Pulse Code Modulation (PCM)
    6. Encoding Digital Data as Digital Signals
    7. Line Coding Schemes: NRZ, RZ, Manchester, AMI
    8. Block Coding, Scrambling
    9. Digital Modulation: Amplitude Shift Keying (ASK), Frequency Shift Keying (FSK), Phase Shift Keying (PSK), Quadrature Amplitude Modulation (QAM)

  6. Multiplexing and Spreading [6 hours]
    1. Multiplexing and Application
    2. Frequency Division Multiplexing (FDM), Wavelength-Division Multiplexing (WDM)
    3. Time Division Multiplexing (TDM)
    4. Spread Spectrum
    5. Code-Division Multiple Access (CDMA)

  7. Switching [3 hours]
    1. Switching and Application
    2. Circuit Switching and Packet Switching
    3. Datagram Switching and Virtual Circuit Switching
    4. X.25, Frame Relay, ATM

  8. Information Theory and Coding  [8 hours]
    1. Introduction to Information Theory, Average Information
    2. Source Coding – Huffman Coding
    3. Error Detection and Correction Codes
    4. Hamming Distance
    5. Linear Block Coding
    6. Cyclic Codes, CRC
    7. Convolution Codes

Practical:

  1. Signal analysis using MATLAB
  2. Bandwidth analysis of different signals using spectrum analyzer
  3. Analog Modulation Generation and Reconstruction
  4. Pulse Modulation Generation and Reconstruction
  5. Conversion of given binary sequence into different line coding
  6. Digital Modulation (ASK, FSK, PSK) Generation and Reconstruction

References:

  1. Data and Computer Communications, Eight Edition, William Stallings
  2. Data Communications and Networking, Fourth Edition, Behrouz A Forouzan
  3. Signals and Systems, A. V. Oppenheim, Latest Edition
  4. Computer Networks, A. S. Tanenbaum, Latest Edition
